## Question 1: Dedupe the array of objects by a property

Dedupe the array based on the id.

```ts
const arr = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
  { id: 1, name: 'Alice again' },
]
```

## Question 3: Create map from them

```ts
// (1) Array with key
const arr = [
  ['a', 1],
  ['b', 2],
  ['c', 3],
]

// (2) Array on an object
const arr = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
]

// (3) Array of an string
const arr = ['x', 'y', 'z']
```
